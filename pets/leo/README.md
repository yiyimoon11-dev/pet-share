# Leo

`Leo` is a custom Codex desktop pet based on a cream-colored chibi mascot with oversized orange-lined ears, glossy blue eyes, a curled tail, and small purple wing-like accents.

## Files

- `pet.json`: Codex pet manifest
- `spritesheet.webp`: animated pet atlas

## Install

1. Copy this `leo` folder into `C:\Users\<your-user>\.codex\pets\`.
2. Restart Codex Desktop.

## Notes

- `pet.json` and `spritesheet.webp` must stay in the same folder.
- If Codex does not detect the pet, check that `pet.json` is UTF-8 without BOM.
