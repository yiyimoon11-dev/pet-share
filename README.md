# pet-share

codex宠物分享。

![Leo preview](./assets/leo-preview.png)

## Included pets

- `Leo`: a custom Codex desktop pet

## Install

1. Download the `pets/leo` folder from this repository.
2. Place it under your local Codex pets directory.
3. Restart Codex Desktop so it rescans custom pets.

### Windows path

`C:\Users\<your-user>\.codex\pets\leo`
